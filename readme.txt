Plugin Name : Contributors

Usage:
Add Contributors to posts, pages and custom post types

How to use:
- Copy the plugin folder to wordpress plugin directory (wp-content/plugins)
- Activate the plugin from the admin's Installed plugins page

Enable Contributors:
- Activation of the plugin will enable the contributors box while editing any post, page or custom post types
- Select contributors from the list and save the post

List Contributors:
- The post with save contributors will show the contributors box in the post's page
- Gravatar profile is linked with the contributors
